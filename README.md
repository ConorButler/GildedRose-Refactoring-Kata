# Gilded Rose

This is the [Gilded Rose](https://github.com/emilybache/GildedRose-Refactoring-Kata) kata in TypeScript.

## Getting started

Install dependencies

```sh
npm install
```

## Running tests

To run all tests

### Jest way

```sh
npm test
```

To run all tests in watch mode

```sh
npm test:watch
```
